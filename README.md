# Cardmarket Ultimate (Free)

A free Chrome extension that enhances Cardmarket Want Lists.

## Features

- Displays card images directly in Want Lists
- Improves readability
- Works automatically on Cardmarket

## Installation

1. Download this repository as ZIP.
2. Extract the ZIP.
3. Open Chrome.
4. Go to:

chrome://extensions

5. Enable **Developer Mode**.
6. Click **Load unpacked**.
7. Select the extracted folder.

The extension is now active.

## Updating

When a new version is released:

1. Download the latest ZIP.
2. Replace the old folder.
3. Click **Reload** in Chrome Extensions.
